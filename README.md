# Automation prototype: automate ipv using image recognition

## Create maven skeleton from an archetype

`mvn archetype:generate 
-DartifactId=ipv 
-DinteractiveMode=false 
-DgroupId=stackpath 
-DarchetypeArtifactId=maven-archetype-quickstart`

## Create .gitignore

1. `cd <desired_folder>`
2. `touch .gitignore`
3. `*
!.gitignore`
